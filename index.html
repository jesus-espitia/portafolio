Mis Proyectos

Animacion de auto bus

La animación del autobús fue creada en Visual Studio Code con HTML, CSS y JavaScript, simulando su movimiento real mediante transiciones y animaciones dinámicas.

Entre Mar y Tierra

Diseño y maquetación de la página web del proyecto Entre Mar y Tierra. fusionando estética visual y narrativa digital para crear una experiencia inmersiva, moderna y coherente con la identidad del proyecto.

Ilustracion con heramienta de diseño

Creación de un oso polar animado utilizando tecnologías como SVG, JSON, HTML y JavaScript. El propósito del proyecto es demostrar conocimientos técnicos en el uso e integración de estas herramientas.

Modelado 3D

Modelado 3D desarrollado en Blender, enfocado en la creación detallada de formas, texturas y proporciones realistas. El proyecto demuestra dominio de las herramientas de modelado, iluminación y renderizado, logrando una representación visual precisa y de alta calidad.

Animacion bidimensional

Se presenta una escena desarrollada a partir de un proyecto ilustrado en Adobe Illustrator, animado en After Effects y estructurado en Visual Studio Code. Este proceso integra diseño gráfico, animación y programación para crear una composición visual dinámica, coherente y técnicamente sólida.

Escuala de comida

La aplicación Escuela de Cocina fue desarrollada con el propósito de instruir y enseñar a nuevos chefs en la preparación de platos exquisitos.

App Comida

App Comida es un prototipo de una tienda de comida rápida que permite realizar pedidos de manera virtual, ofreciendo una experiencia práctica y moderna para el usuario.

Video Mario Bros

Animación inspirada en el universo de Mario Bros, desarrollada en Adobe After Effects. El proyecto combina técnicas de animación digital, efectos visuales y sincronización de movimiento para recrear una secuencia dinámica y atractiva, destacando la creatividad y dominio del software en la producción audiovisual.
